# Accessibility Checklist

- [ ] Page has one clear `h1`.
- [ ] Main navigation links are reachable by keyboard.
- [ ] Buttons have visible text or `aria-label`.
- [ ] Form fields have labels.
- [ ] Text contrast is readable.
- [ ] 375px viewport has no horizontal overflow.
- [ ] 768px viewport displays correctly.
- [ ] 1440px viewport displays correctly.
- [ ] Motion is reduced when `prefers-reduced-motion` is enabled.
- [ ] Images include meaningful `alt` text or empty `alt` when decorative.
