# Publishing Checklist

- [ ] Lesson uses only authorized text, original text, or properly licensed excerpts.
- [ ] No full copyrighted article or full lyrics are included.
- [ ] WordPress post title is clear.
- [ ] Category and tags match `category-map.json`.
- [ ] Vocabulary table is readable on mobile.
- [ ] Answer key uses `details` and `summary`.
- [ ] YouTube is embedded with the official WordPress block only.
- [ ] Interactive GitHub Pages link works.
- [ ] Preview is checked on phone and desktop.
- [ ] Publishing is manual.
