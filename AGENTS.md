# Jasper Language Lab Agent Rules

1. 預設只建立草稿與可預覽內容。
2. 不得自動發布到 WordPress。
3. 不得刪除 WordPress 文章或媒體。
4. 不得把密碼、Token、Cookie 或 API Key 加入 repository。
5. 不得假設擁有 WordPress 管理權限。
6. 優先使用 WordPress 原生區塊。
7. 所有介面必須 mobile first。
8. 所有頁面必須支援繁體中文。
9. 程式必須可在 GitHub Pages 執行。
10. JavaScript 錯誤不能阻止文章閱讀。
11. 所有互動按鈕需要可辨識的 aria-label。
12. 顏色對比需符合基本 WCAG 可讀性。
13. 不得使用受版權保護的完整文章或完整歌詞作為示範內容。
14. 修改前先閱讀 README.md 和 AGENTS.md。
15. 每次工作完成後，列出：
    - 修改的檔案
    - 完成的功能
    - 尚未完成的功能
    - 測試結果
    - 下一步建議
